# Learning SaaS - System Zarządzania Nauką

Projekt realizowany w ramach zajęć: **Architekt Biznesu SaaS**.
Prowadzący: **Krzysztof Saja**.

---

## 🚀 O Projekcie
**Learning** to nowoczesna platforma SaaS zaprojektowana, aby pomóc użytkownikom w ustrukturyzowanej nauce nowych umiejętności. Łączy zarządzanie materiałami z analityką postępów.

## 🛠️ Stack Technologiczny
Aplikacja została zaprojektowana przy użyciu następujących technologii:
* **Frontend:** React JS, Tailwind CSS, Shadcn UI
* **Backend/Baza danych:** Supabase (PostgreSQL)
* **Autentykacja:** Supabase Auth
