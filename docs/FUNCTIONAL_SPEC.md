# FUNCTIONAL_SPEC.md

## Priorytet Must-have
1. Autoryzacja i sesje:
   - rejestracja/login (email+hasło)
   - reset hasła
2. Ścieżki nauki:
   - tworzenie/edycja/usuwanie ścieżek
   - dodawanie modułów i lekcji
   - oznaczanie postępów
3. Kalendarz nauki:
   - planowanie sesji i zadań
   - przypomnienia i powiadomienia
4. Statystyki:
   - dzienne / tygodniowe raporty
   - wykresy czasu nauki i ukończenia
5. Integracja danych:
   - przechowywanie zasobów w Supabase Storage
   - synchronizacja z postępami w kursie

## Priorytet Should-have
1. Współpraca grupowa:
   - udostępnianie ścieżek i notatek
2. Wersje mobile-friendly i PWA
3. Rozszerzone dashboardy mentorów (B2B):
   - metryki zespołowe, alerty obniżonej aktywności
4. Automatyczne rekomendacje kursów w oparciu o postęp i cele
5. Tryb offline-first z lokalnym cache
