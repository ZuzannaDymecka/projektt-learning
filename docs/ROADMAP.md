# ROADMAP.md

## Faza MVP (0-3 miesiące)
- podstawowy login / rejestracja (Supabase Auth)
- CRUD ścieżki, lekcji, progresu
- prosty kalendarz z przypomnieniami
- dashboard statystyk (tydzień, miesiąc)
- responsywny UI w Tailwind + Shadcn

## Faza v1.0 (3-6 miesięcy)
- obsługa materiałów w Supabase Storage
- zaawansowane filtry i cele nauki
- współpraca zespołowa i udostępnianie ścieżek
- raporty B2B dla menedżerów

## Faza v2.0 (6-12 miesięcy)
- AI rekomendacje ścieżek i personalizacji
- integracje z zewnętrznymi LMS/Calendar API
- tryb offline + PWA
- rozbudowany moduł quizów i certyfikatów
