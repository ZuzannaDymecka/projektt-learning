# USER_STORIES.md

## Uczeń
- Jako uczeń chcę zarejestrować się i zalogować, aby mieć spersonalizowany dostęp do moich ścieżek nauki.
- Jako uczeń chcę dodawać lekcje do ścieżki, aby móc planować swoje etapy nauki.
- Jako uczeń chcę widzieć prognozę czasu potrzebną do ukończenia kursu, aby lepiej organizować dni.

## Mentor
- Jako mentor chcę monitorować postępy przypisanych podopiecznych, aby szybko wychwycić problemy.
- Jako mentor chcę wysyłać przypomnienia do grupy, aby zwiększyć frekwencję na sesjach.

## Zespół HR (B2B)
- Jako menedżer chcę eksportować dane z postępów, aby raportować do działu HR.
- Jako menedżer chcę definiować role i poziomy dostępu, by kontrolować poufność danych.
